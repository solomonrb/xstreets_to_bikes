NYC Citibike Stations via SMS

Text your current cross streets or address, and you will receive a response with the 3 closest Citibike stations and their bike/dock availability.

With a Citibike key and a dumb phone, you no longer need the Citibike app (or a smartphone)!

To opt in, simply text the number:

111-111-1234

You can opt out at any time by sending a "UNSUB" message (without the quotes) to the same number.

Texting the number again will opt you back in.

You will NEVER receive an unprompted message, only a message in response to the message you send.

This is an open source hobby project. Check out the GitHub repository.
